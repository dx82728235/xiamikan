# 📺 虾米看

> 全平台聚合、免广追剧与原生 DLNA 投屏神器 🚀

**虾米看** 是一款基于 UniApp 开发的移动端视频聚合应用。它将爱奇艺、腾讯视频、优酷三大平台的移动端网页无缝整合，并内置了强大的智能解析引擎、底层视频流嗅探器以及物理级防劫持去广告系统，为用户提供纯净、流畅、可投屏的终极观影体验。

## ✨ 核心特性

* 🌐 **多平台自由无感切换**
  * 原生 ActionSheet 菜单，一键在爱奇艺、腾讯、优酷之间无缝横跳。
* 🚀 **全天候智能解析引擎**
  * **精准狙击**：突破手机端复杂单页应用 (SPA) 限制，自动挖掘网页底层变量。
  * **智能拼装**：精准提取优酷 `vid` / `id_X...` 以及腾讯 `cid` / `vid`，强行组装出 100% 纯正的电脑版 URL 交由接口解析。
* 📺 **原生 DLNA 深度投屏**
  * 放弃脆弱的网页投屏，采用底层网络层资源监控 (`window.performance`)。
  * 完美嗅探真实 `m3u8` / `mp4` 数据流，支持 iframe 隔离层穿透，一键发送至局域网智能电视。
* 🛡️ **重装装甲：物理级去广告防线**
  * **防弹窗劫持**：拦截全局 `window.open`。
  * **防误触跳转**：全局拦截非 `<video>` 标签的 `click` 事件，切断流氓 `<a>` 链接跳转。
  * **视觉大扫除**：高频定时器持续注入脚本，自动击碎并隐藏所有高层级 (`z-index > 99`) 的绝对定位悬浮窗广告。
* 🔄 **硬件级强制横屏**
  * 摆脱网页全屏的伪横屏限制，调用底层 API (`plus.screen.lockOrientation`) 强行控制硬件陀螺仪，实现真正的沉浸式观影体验。
* 🕒 **无痕本地观看历史**
  * 解析瞬间自动截获真实标题与链接，加密写入手机本地存储 (`Storage`)。
  * 支持最多 20 条记录循环覆盖，提供一键续播与“一键清空”隐私保护功能。

## 🛠️ 技术栈

* **框架**: [UniApp](https://uniapp.dcloud.net.cn/) / Vue.js
* **跨平台 API**: HTML5+ (`plus.nativeObj.View`, `plus.screen`, `plus.nativeUI`)
* **核心组件**: `<web-view>` 容器深度交互与双向通信 (`evalJS`, `titleUpdate`)

## 📦 安装与运行指南

### 1. 环境准备
* 下载并安装 [HBuilderX](https://www.dcloud.io/hbuilderx.html) 开发者工具。
* 准备一台 Android 手机或模拟器，并开启“开发者模式”和“USB 调试”。

### 2. 导入项目
* 将本项目克隆或下载到本地。
* 将文件夹直接拖入 HBuilderX 中。

### 3. 核心配置 (重要)
* 打开项目根目录的 `manifest.json`，进入**App常用其它设置**。
* 勾选 **“支持屏幕旋转”**（否则横屏功能失效）。
* 确保已配置 DLNA 投屏相关的原生插件（如 `JX-Dlna`）。

### 4. 编译运行
* 点击 HBuilderX 顶部菜单栏：`运行` -> `运行到手机或模拟器` -> 选择你的设备。
* 如需生成最终安装包，点击：`发行` -> `原生App-云打包`，选择“使用公共测试证书”打包 `.apk`。
